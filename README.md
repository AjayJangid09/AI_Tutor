# AI_Tutor
Built a production-grade open-source RAG AI Tutor that ingests PDFs/text and uses hybrid retrieval (Dense+BM25+RRF+reranker) with Qwen2.5 for grounded answers. Added guardrails (PII masking, injection &amp; toxicity checks, NLI hallucination filter), multi-turn memory, evaluation suite, and detailed logging for reliability.
